# Site Service — Главный веб-интерфейс

Фронтенд платформы CheckDev.

## 🔧 Технологии
- Java 21
- Spring Boot 2.7.11
- Thymeleaf + Bootstrap
- Spring WebFlux (WebClient)
- Eureka Client

## 📦 Порт
8080

## 🌐 Доступ
http://localhost:8080

## 📚 Страницы сайта

| URL | Описание |
|-----|----------|
| / | Главная |
| /categories/ | Категории |
| /topics/{id} | Темы категории |
| /topic/{id} | Детали темы |
| /interviews/ | Список собеседований |
| /interview/{id} | Детали собеседования |
| /interview/createForm | Создание собеседования |
| /profiles/ | Профили пользователей |
| /profiles/{id} | Профиль пользователя |
| /persons/ | Мой профиль |
| /messages/ | Сообщения |
| /login | Авторизация |
| /registration | Регистрация |

## 🔗 Взаимодействие с сервисами (через Eureka)

| Сервис | Порт | Использование |
|--------|------|---------------|
| auth | 9900 | Аутентификация, профили |
| desc | 9902 | Категории, темы |
| mock | 9912 | Собеседования, участники |
| generator | 9903 | Генерация вопросов |
| notification | 9920 | Уведомления, подписки |

## 🔗 Eureka
Регистрируется под именем site

## 🚀 Запуск всех сервисов

1. **Eureka** (порт 9009) — сначала!
2. **auth** (9900)
3. **desc** (9902)
4. **mock** (9912)
5. **generator** (9903)
6. **notification** (9920)
7. **site** (8080) — последним

```bash
mvn spring-boot:run
